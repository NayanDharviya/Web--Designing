PROJECT

  • Title : GST Billing System

  • Environment :

    • Front-End : Visual C#

    Back-End : SQL Server 2008

    Team Size : 2

    Duration : 6 months

    Role : Designer, Developer, Tester

    Description : GST Billing System is a desktop application. This System provides the Details of all Customers, Suppliers and Products. And also generates the Reports like Bill report, item wise sale/purchase report, Customer sale Report (date/month/year) wise. And it also allow the user to find GST Amount (month/year/day) wise .

    Objective

    1. To reduce the paper work.

    2. To reduce the manual work.

    3. To improve and easily maintain the business.

    4. Accurate result can be obtained.
